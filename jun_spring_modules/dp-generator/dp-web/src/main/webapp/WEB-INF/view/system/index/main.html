基本信息

   项目介绍

  • 一个能快速生成代码的开发脚手架
  • 命名规范和工程分层规约参考阿里巴巴JAVA开发规范
  • 自动生成sql、dao、manager、service、controller、html、js，更快的开发方式
  • 生成代码可直接用于dp-LTE和dp-PRO开源项目
  • 支持多模块配置，可快速生成个性化业务模块基础代码和结构
