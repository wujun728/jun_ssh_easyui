# rbac-layui

#### 介绍
使用基本技术栈
    
- spring boot
- spring boot web
- spring boot cache
- spring boot session
- spring boot security
- spring boot data redis
- mybatis
- pagehelper
- druid
- freemarker


前端主要使用了 layui  UI框架和  layui-mini 后台模板
